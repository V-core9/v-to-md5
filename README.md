🔩 Install using command : 

    npm install v_to_md5 --save

🪁 How to use: 
  
    const v_to_md5 = require("v_to_md5");

    console.log( v_to_md5("demo_password_123456") )
    
//-> returns hash : b9a0dc00aff156a2a5e915ba00a590ba
